Conceptos básicos

Aplicaciones Moviles o APP

Son aplicaciones diseñada para ser ejecutadas en teléfonos inteligentes, tablets y otros dispositivos móviles y que permite al usuario efectuar una tarea, facilitando la gestion o actividad a desarrollar.

Ejemplos de Aplicaciones Móviles

  • Whatsap
  • Telegram
  • Instagram
  • Opera Mini
  • Clash of Clans

Lenguajes para desarrollar APP

iOS Android Windows Phone
  • ObjetiveC
  • Java
  • .NET
  • Visual Studio

Ventajas de las Aplicaciones Webs

  • Facilidad de acceso (sólo se necesita navegador web)
  • Independencia de sistema operativo
  • Facilidad de actualización y mantenimiento (sin redistribuir o instalar software)

Aplicaciones Nativas

Se desarrolla de forma específica para un determinado sistema operativo, llamado Software Development Kit o SDK.

Aplicaciones Nativas

Ventajas Desventajas
  • Acceso completo al dispositivo
  • Mejor experiencia del usuario
  • Visibilidad en APP Store
  • Envio de notificaciones a los usuarios
  • La actualización es constante
  • Diferentes habilidades/idiomas/herramientaspara cada plataforma de destino
  • Tienden a ser más caras de desarrollar
  • El código cliente no es reutilizable entre las diferentes plataformas

Web APP

Se ejecutan dentro del propio navegador web del dispositivo a través de una URL.

Web APP

Ventajas Desventajas
  • Se puede distribuir por tiendas iOS y Android
  • Instalación nativa, pero construida con JavaScript, HTML y CSS
  • Código base para múltiples plataformas
  • Acceso a parte del hardware del dispositivo
  • Experiencia del ususario más propia de la aplicación web que de la app nativa
  • Diseño visual no siempre relacionado con el sistema operativo en el que se muestre

Aplicación híbrida

Es una combinación de las aplicaciones nativas y las webapss. Se desarrollan con lenguajes: HTML, Javascript y CSS por lo que permite su uso en diferentes plataformas y el acceso a gran parte de las características del hardware del dispositivo. Es posible distribuirla en app store.

Apache Cordova

Es un framework para desarrollar aplicaciones nativas dentro de un smartphone. Cuenta con muchas Apis de diversos dispositivos móviles. Permite desarrollar con HTML, JavaScript y CSS

Diseño Respondivo

Diseño web adaptable, donde la apariencia de las páginas web se adapta al dispositivo que se esté utilizando para visualizarla.

Bootstrap

Es un framework desarrollado por Twitter para crear interfaces que se adapten al tamaño de pantalla de diversos dispositivo.
Está basado en HTML5 y CSS3 por lo que es compatible con la mayoría de los navegadores web.
Contiene plantillas de diseño con tipografía, formularios, botones, cuadros, menús de navegación y otros elementos de diseño y requiere la librería jQuery para que todos los plugins JavaScript funcionen.
